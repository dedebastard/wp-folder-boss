== Description ==

**WP Folder Boss** is a complete folder organizer for WordPress. Organize every area of your WordPress admin using a hierarchical folder system with drag-and-drop support, bulk move, and auto-assignment of new uploads.

= Features =

* **Media Library Folders** — Add folders to the Media Library (grid and list views).
* **Post Type Folders** — Organize Pages, Posts, and any Custom Post Type.
* **WooCommerce Support** — Folders for Products, Orders, and Coupons.
* **Users & Plugins** — Organize Users and installed Plugins into folders.
* **Drag & Drop** — Native HTML5 drag-and-drop to move items into folders.
* **Bulk Move** — Move multiple items at once using bulk actions.
* **Auto-Assign Uploads** — New uploads go into the currently selected folder.
* **Folder Tree Sidebar** — Expandable, hierarchical folder tree with item counts.
* **Right-click Context Menu** — Create, rename, delete folders from a context menu.
* **REST API** — Full REST API for all folder operations.
* **Import Tool** — Import folders from FileBird or Real Media Library.

== Installation ==

1. Upload the `wp-folder-boss` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen in WordPress.
3. Go to **Settings → WP Folder Boss** to configure which screens to enable.

== Frequently Asked Questions ==

= Does this work with WooCommerce? =

Yes! WP Folder Boss automatically detects WooCommerce and adds folder support to Products, Orders, and Coupons.

= Can I import folders from FileBird? =

Yes. Go to **Settings → WP Folder Boss → Import** and select your source plugin.

= Does it support custom post types? =

Yes. Any registered public post type can be enabled in the settings.
